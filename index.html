What It Is

An interactive web tool that calculates the exact solar system size needed based on appliance list and energy consumption.

How to Use

  1. Bookmark the link to your browser
  2. When a homeowner sends their appliance list, open the calculator
  3. Input each appliance, quantity, and daily usage hours
  4. The calculator shows: daily consumption, peak load, recommended inverter size, battery capacity, and panel count
  5. Copy these numbers into your WhatsApp messages

Benefits

  • ✓ Eliminates manual calculations (saves 10+ minutes per lead)
  • ✓ Ensures system sizing is accurate and defensible
  • ✓ Reduces over-sizing and under-sizing errors
  • ✓ Creates confidence through data-driven recommendations
  • ✓ Ties ROI numbers to actual system specifications

Why It Was Created

Most installers guess at system sizes, leading to either oversized systems (wasting money) or undersized systems (poor performance).

Impact

Accurate sizing = accurate ROI numbers = confident homeowners
